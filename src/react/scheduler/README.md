# `scheduler`

This is a package for cooperative scheduling in a browser environment. It is currently used internally by React, but we plan to make it more generic.
这是一个用于浏览器环境中的协作调度的包。React 目前在内部使用它，但我们计划使它更通用。
此软件包的公共 API 尚未最终确定。
